# AI Email Intelligence Platform

AI Email Intelligence is an AI-powered email productivity platform that helps users prioritize emails, analyze attachments, summarize threads, generate replies, and manage follow-ups.

## Key Features

- AI email priority scoring
- Risk and intent detection
- Gmail Primary + useful Spam recovery
- Attachment intelligence for PDF, DOCX, images, certificates, contracts, invoices, resumes, and offer letters
- OCR and document understanding
- AI-generated reply drafts
- Multi-reply generation
- Thread summary
- Follow-up and reminder workflow
- Analytics dashboard
- Light/dark UI

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- FastAPI
- Python
- Gmail API
- SQLite
- Ollama / OpenAI
- OCR tools
